# Приложение 2. Алгоритм расчёта critical slowing down (CSD)

**Входные данные:** временной ряд ΔE (измерения каждые 4–6 ч в течение 24–72 ч) или t<sub>корковый</sub> (для сенсорного ΔE).

**Шаги расчёта:**

1. Формирование скользящего окна из 6–12 последовательных измерений.
2. Расчёт автокорреляции первого порядка (AR(1)) для каждого окна:

   AR(1) = \frac{\sum_{i=1}^{n-1} (x_i - \bar{x})(x_{i+1} - \bar{x})}{\sum_{i=1}^{n} (x_i - \bar{x})^2}

   где x_i — значение ΔE в момент i, \bar{x} — среднее по окну.

3. Расчёт дисперсии ΔE для каждого окна.
4. Определение тренда:

   - AR(1) > 0,7 **и** рост дисперсии > 30 % за 24 ч → сигнал тревоги (t<sub>crit</sub>).

**Примечание:** приведённые пороги являются иллюстративными. Фактические значения должны быть определены в ходе валидационных исследований.

**Визуализация:** график ΔE с наложенными AR(1) и дисперсией.
