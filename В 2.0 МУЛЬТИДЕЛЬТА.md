## Аннотация

### Универсальная координатная модель пульсации (Φ, Ψ, E, G)

Фреймворк МУЛЬТИДЕЛЬТА задаёт динамическую координатную систему для оценки жизнеспособности систем с обратной связью.

**Координаты:**
- Φ — ресурс устойчивости
- Ψ — нагрузка / задержка отклика
- E — интегральное отклонение
- G = dE/dt — направление движения (восстановление / деградация / пульсация)

**Универсальность:** параметры Φ, Ψ, K_GB могут быть переопределены под конкретную систему (нервную, сердечно-сосудистую, онкологическую, клеточную). Критерий знака G < 0 (восстановление) / G > 0 (деградация) остаётся инвариантом.

**Пример реализации в данном документе:** модель адаптирована к нейрофизиологии (SSEP, HRV, gut-brain axis). Для нервной системы эмпирические пороги и коэффициенты уже частично определены. Для других областей требуется калибровка α, β, σ.

**Статус:** фреймворк является гипотезой, требующей клинической валидации в каждой области применения.

## Введение

**Проблема.** В реанимации, нейрохирургии и неврологии отсутствует объективный количественный показатель, отражающий жизнеспособность нервной системы в реальном времени.

МРТ фиксирует повреждение постфактум. ЭНМГ и вызванные потенциалы дают разрозненные параметры. Клинический осмотр у пациента в коме или под седацией ограничен.

**Решение.** Фреймворк «МУЛЬТИДЕЛЬТА» сводит разнородные данные к двум координатам:

- Φ — запас компенсаторных возможностей (ресурс устойчивости)
- Ψ — нагрузка / задержка отклика регуляторных систем

Интегральный индекс E отражает удаление системы от устойчивого состояния. Производная G = dE/dt отражает направление динамики.

## 1. Доказательная база

### 1.1 Реанимация (пост-аноксическая кома)

- N20-P25 < 0.64 μV (12–72 ч после ROSC): specificity 100%, FPR 0% (Oh et al., 2019)
- N20-P25 ≤ 0.38–1.01 μV (12–72 ч): FPR 0%, sensitivity 61–82% (van Soest et al., 2021)
- N20-P25 < 0.4 μV (48–72 ч): неизбежно неблагоприятный исход, FPR 0% (Glimmerveen et al., 2020)

Билатеральное отсутствие N20 — сильный предиктор утраты жизнеспособности корковых механизмов.

### 1.2 Нейроонкология

Сохранённые или улучшающиеся параметры SSEP коррелируют с лучшей нейропластичностью и функциональным исходом (Huang et al., 2025).

### 1.3 Нейропластичность

- Yoon et al. (2021): корреляция SSEP с восстановлением функций
- Ma et al. (2011): уменьшение временной дисперсии
- Arciniegas-Villanueva et al. (2022): N70 как маркер кортико-кортикальной пластичности

### 1.4 Gut-brain axis

Бутират и Akkermansia muciniphila снижают нейровоспаление и стабилизируют гематоэнцефалический барьер. Вводится показатель метаболической модификации:

K_GB = [Butyrate] / [LPS]

## 2. Координатная модель

### 2.1 Функция деградации проводимости

g(ΔE) = 1 / (1 + exp((ΔE - 6.5)/σ)), σ ≈ 0.5

При ΔE ≥ 6.5 компенсаторный резерв экспоненциально подавляется (асимптотически стремится к нулю).

### 2.2 Координаты

Φ = (A_N20-P25 / 2.31) * g(ΔE) * K_GB

Ψ = (ΔE / 6.5) * (1 + (40 - RMSSD)/40) * (1 / K_GB)

**Интерпретация:**
- A_N20-P25 — амплитудный резерв (мкВ)
- ΔE — временная деградация проводимости (мс)
- RMSSD — вариабельность сердечного ритма (мс)
- K_GB — метаболический модификатор (нормирован)

### 2.3 Интегральный индекс

Регуляризационная константа ε = 0.1 (предотвращает сингулярность при Φ → 0).

E = α * 1/(Φ + ε) + β * Ψ

α и β — калибруемые веса. Начальное приближение: α = β = 1.

---

## 3. Динамическая интерпретация

### 3.1 Теоретическая форма

G = ∇E · F = dE/dt

### 3.2 Практическая оценка

В силу дискретности измерений используется численная оценка:

G(t) ≈ (E(t) - E(t-Δt)) / Δt

### 3.3 Критерий знака

- G < 0 → восстановление (система возвращается)
- G ≈ 0 → граница (пульсация)
- G > 0 → деградация (система уходит)

Смена знака G во времени — признак пульсационного режима.

### 3.4 Масштабы времени

Параметры имеют различную динамику:
- SSEP — минуты–часы
- HRV — секунды–минуты
- K_GB — часы–сутки

При расчёте используется последняя доступная синхронизированная оценка параметров.

### 3.5 Область корректности

Интерпретация G корректна при квазистационарности входных параметров на интервале Δt.

### 3.6 Алгоритм мониторинга

На каждом шаге Δt:
1. Рассчитать Φ(t), Ψ(t)
2. Рассчитать E(t)
3. Оценить G(t) по конечной разности
4. Классифицировать состояние по знаку G

---

## 4. Архитектурная связь с общим законом

Фреймворк является частным случаем закона архитектурной согласованности (доказательство неизбежности пульсации — в работе «Квантовая пульсация 5.2»).

Φ ↔ Φ, Ψ ↔ Ψ, E ∼ E_откл, G ∼ ∇E · F

**Режимы:**

- Φ > 1, Ψ < c0, G < 0 → устойчивость (восстановление)
- Φ ≈ 1, Ψ ≈ c0, знак G меняется → пульсация
- Φ < 1 или Ψ ≥ c0, G > 0 → разнос (деградация)

Эмпирические пороги для нейрореанимации: Φ > 1.2, Ψ < 0.65, E < 2.0 — зелёная зона.

## 5. Примеры расчёта

### 5.1 Критическое состояние (пост-аноксическая кома)

Входные данные: A_N20-P25 = 1.8 мкВ, ΔE = 8.2 мс, RMSSD = 18 мс, K_GB = 0.75

g = 1 / (1 + exp((8.2 - 6.5)/0.5)) ≈ 0.03
Φ = (1.8 / 2.31) * 0.03 * 0.75 ≈ 0.017
Ψ = (8.2 / 6.5) * (1 + (40-18)/40) * (1 / 0.75) ≈ 2.60
E = 1 * 1/(0.017 + 0.1) + 1 * 2.60 ≈ 11.15

При E(t-Δt) = 10.0: G > 0 → деградация.

### 5.2 Состояние восстановления

Входные данные: A_N20-P25 = 2.9 мкВ, ΔE = 6.4 мс, RMSSD = 28 мс, K_GB = 0.95

g = 1 / (1 + exp((6.4 - 6.5)/0.5)) ≈ 0.55
Φ = (2.9 / 2.31) * 0.55 * 0.95 ≈ 0.66
Ψ = (6.4 / 6.5) * (1 + (40-28)/40) * (1 / 0.95) ≈ 1.34
E = 1 * 1/(0.66 + 0.1) + 1 * 1.34 ≈ 2.65

При E(t-Δt) = 2.8: G < 0 → восстановление.

## 8. Методология валидации и статистический анализ

### 8.1 Дизайн исследования

**Тип исследования:** проспективное многоцентровое когортное исследование.

**Конечные точки:**
- Первичная: функциональный исход через 3 и 6 месяцев (GOS-E, CPC)
- Вторичные: динамика Φ, Ψ, E, G в процессе мониторинга

**Критерии включения:**
- Пациенты в коме после остановки кровообращения (12–72 ч после ROSC)
- Пациенты после нейрохирургических операций с риском отёка/ишемии
- Пациенты с тяжёлой ЧМТ

**Критерии исключения:**
- Технически некачественная регистрация SSEP
- Выраженная гипотермия (температура < 34°C)
- Применение миорелаксантов в момент регистрации

### 8.2 Протокол измерений

**Частота регистрации:**
- SSEP: 2 раза в сутки (утро/вечер) в первые 5 дней
- HRV: непрерывно, усреднение за 5-минутные окна
- K_GB: 1 раз в сутки (утро)

**Условия регистрации:**
- Температура кожи в месте стимуляции 32–34°C
- Импеданс электродов < 5 кОм
- Частота дискретизации ≥ 2000 Гц
- Усреднение не менее 200 эпох на точку

**Контроль качества:**
- Две независимые регистрации на каждом временном срезе
- Слепой анализ (врач, рассчитывающий Φ, Ψ, E, G, не знает исход)

### 8.3 Методы обработки данных

**Предобработка:**
- Полосовая фильтрация SSEP: 5–2000 Гц
- Режекция артефактов (мышечные, электромагнитные) вручную или автоматически
- Интерполяция пропущенных значений (линейная, при пропусках < 20%)

**Расчёт параметров:**
- A_N20-P25: автоматическое определение пиков с ручной верификацией
- ΔE: разность латентностей N20/P40 и M-ответа
- RMSSD: расчёт по 5-минутным интервалам RR
- K_GB: по данным лабораторных анализов (бутират, LPS)

### 8.4 Статистический анализ

**Описательная статистика:**
- Среднее, медиана, стандартное отклонение, межквартильный размах для Φ, Ψ, E, G
- Стратификация по исходам (благоприятный/неблагоприятный)

**ROC-анализ:**
- Построение ROC-кривых для E и G на 1, 3, 5 сутки
- Определение пороговых значений (максимальная чувствительность + специфичность)
- Сравнение AUC с существующими методами (N20-P25, билатеральное отсутствие)

**Регрессионный анализ:**
- Логистическая регрессия: P(неблагоприятный исход) = f(Φ, Ψ, E, G, возраст, пол, этиология)
- Cox-регрессия для анализа времени до восстановления

**Калибровка α, β, σ:**
- Поиск оптимальных α, β, σ на тренировочной выборке (70% данных)
- Валидация на тестовой выборке (30% данных)
- Критерий: максимизация AUC и калибровочного наклона

**Анализ корреляций:**
- Оценка корреляции между ΔE и другими параметрами (Spearman)
- Факторный анализ для проверки независимости Φ и Ψ

**Размер выборки:**
- Для ROC-анализа: не менее 50 событий на предиктор
- При 3–5 предикторах — не менее 150–250 пациентов

### 8.5 План валидации

**Этап 1. Пилотное исследование (1 центр, n = 30–50)**
- Оценка осуществимости
- Предварительная калибровка α, β, σ
- Уточнение протокола

**Этап 2. Основное исследование (3–5 центров, n = 150–250)**
- Валидация порогов
- Оценка прогностической ценности
- Калибровка вероятностной модели

**Этап 3. Внешняя валидация (независимый центр, n = 100)**
- Проверка на новых данных
- Анализ устойчивости коэффициентов

## 9. Тонкие моменты и перспективы калибровки

### 9.1. Асимптотическое поведение Φ

При ΔE ≥ 6.5 резерв экспоненциально подавляется (асимптотически стремится к нулю, но не обнуляется). Для клинического применения необходимо определить порог отсечения.

### 9.2. Производная G: теоретическая и численная

В практической реализации используется конечная разность. Необходимо определить минимальное Δt, при котором численная оценка устойчива к шуму.

### 9.3. Доминирование ΔE

ΔE входит в обе координаты, создавая потенциальную корреляцию между Φ и Ψ. Модель не предполагает независимости координат.

### 9.4. Параметр K_GB

Теоретически обоснован, но клинически трудно измерим в реальном времени. Требует дальнейшего изучения; возможно исключение из быстрой версии протокола.

### 9.5. Отсутствие перехода от G(t) к вероятности исхода

Модель даёт направление движения, но не вероятность исхода. Для клинического использования необходима калибровочная кривая P(неблагоприятный исход) = f(G, Φ, Ψ, E).

## 10. Ограничения применения

### 10.1 Технические ограничения

- SSEP чувствителен к артефактам (электромагнитные помехи, мышечные сокращения)
- Требуется стандартизация оборудования и протоколов
- Невозможность регистрации при нестабильной гемодинамике

### 10.2 Клинические ограничения

- Влияние седативных препаратов на латентности и амплитуды
- Влияние гипотермии (< 35°C) на нервную проводимость
- Двусторонние поражения без возможности калибровки t_базовая

### 10.3 Организационные ограничения

- Требует обученного персонала (нейрофизиолог, реаниматолог)
- Высокая стоимость оборудования для клиник без нейромониторинга
- Необходимость синхронизации ЭЭГ и ЭМГ систем

### 10.4 Статус метода

Метод является **научной гипотезой**, не валидирован клинически. Не может использоваться для принятия клинических решений без подтверждения эффективности в проспективных исследованиях.

## 11. Критические замечания (для рецензентов)

### 11.1 Методологические замечания

**Необходимость детальных протоколов измерений.** Текущая версия протокола содержит базовые требования (частота дискретизации, фильтры, условия регистрации), но требует дальнейшей детализации: точное размещение электродов для разных нервов, алгоритм усреднения и режекции артефактов, порядок действий при технических сбоях.

**Требуется уточнение методов обработки данных.** Обработка SSEP (автоматическое определение пиков) требует валидации на независимых данных. Методы интерполяции пропусков и фильтрации должны быть стандартизованы.

**Недостаточно информации о калибровке параметров.** Коэффициенты α, β, σ требуют калибровки по клиническим данным. Начальное приближение α = β = 1, σ = 0.5 эмпирическое. Валидация должна включать калибровку на тренировочной выборке.

### 11.2 Ответы на критические замечания

**Ограниченная доказательная база для K_GB.** Параметр K_GB включён как теоретический модификатор. Валидация должна определить его клиническую значимость. При невозможности измерения K_GB в реальном времени он может быть исключён из быстрой версии протокола.

**Доминирование ΔE в обеих координатах.** Это не ошибка, а особенность модели. Φ и Ψ не предполагаются независимыми; E и G являются агрегированными индексами. Валидация должна оценить, насколько эта корреляция влияет на прогностическую ценность.

**Отсутствие вероятностной интерпретации G.** Текущая версия даёт направление движения (улучшение/ухудшение). Переход к вероятности исхода — задача следующих версий, требующая калибровки на клинических данных.
