# Приложение 5. Типовой пример резюме для грантовой заявки

**Название проекта:** «Клиническая валидация метода ΔE как маркера функционального состояния нервной системы».

**Гипотеза:** ΔE позволяет количественно оценивать функциональное состояние нервной системы и прогнозировать критические переходы (ухудшение, осложнения) на доклинической стадии.

**Цели:**

1. Определить нормативные значения ΔE и t<sub>базовая</sub> для здоровых лиц (n = 50).
2. Оценить чувствительность и специфичность ΔE у пациентов с радикулопатией (n = 30) до и после декомпрессии.
3. Валидировать алгоритм CSD для прогнозирования послеоперационных осложнений (n = 20).
4. Разработать стандартизированный протокол регистрации и расчёта.

**Методы:**

- регистрация ЭЭГ (N20/P40), ЭМГ (M‑ответ), синхронизированная стимуляция;
- расчёт ΔE и параметров CSD (AR(1), дисперсия);
- сравнение с клиническими исходами и МРТ.

**Ожидаемые результаты:**

- нормативные таблицы ΔE с поправкой на возраст/рост/температуру;
- диагностические пороги ΔE для обратимой/необратимой дисфункции;
- алгоритм CSD с пороговыми значениями AR(1) и дисперсии;
- стандартизированный протокол для клинической практики.
