# Метод ΔE.

### Количественный нейрофизиологический показатель, объединяющий латентности M‑ответа и соматосенсорного вызванного потенциала (N20/P40). Позволяет оценивать функциональное состояние периферических нервов и спинного мозга, а также прогнозировать критические переходы (critical slowing down) на доклинической стадии.

Потенциальное применение:
- предоперационная оценка функционального резерва;
- интраоперационный мониторинг;
- раннее выявление послеоперационных осложнений;
- мониторинг в нейрореанимации и реабилитации.

Метод находится на стадии теоретической разработки и требует экспериментальной и клинической валидации.
