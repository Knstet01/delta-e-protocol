## О проекте

Этот репозиторий содержит протокол измерения **ΔE** — количественного показателя временной задержки между сенсорным сигналом и вербальным ответом в задачах самоописания.

**Цель:**  
Разработать стандартизированный метод объективной оценки степени сохранности связи между телом и сознанием у пациентов с нарушениями сознания, двигательными и когнитивными расстройствами.

**Статус:**  
Гипотеза. Требуется клиническая валидация.
