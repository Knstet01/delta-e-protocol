# Приложение 3. Форма информированного согласия

**Исследование:** оценка функционального состояния нервной системы методом ΔE.

**Цель:** разработка и валидация нового нейрофизиологического метода.

**Процедура:**

- стимуляция нерва короткими электрическими импульсами;
- регистрация ЭЭГ и ЭМГ;
- длительность 20–30 минут, процедура безболезненна.

**Риски:** минимальное раздражение кожи, покраснение в месте наложения электродов (проходит в течение часа). При многократных измерениях возможно лёгкое раздражение; при появлении дискомфорта процедура может быть прервана.

**Преимущества:** вклад в развитие медицины, получение данных о состоянии нервной системы.

**Конфиденциальность:** данные анонимизируются, используются только для научных целей.

**Добровольное участие:** пациент вправе отказаться в любой момент без последствий для лечения.
