# Глоссарий

**ΔE** — количественный показатель функционального состояния нервной системы: разность между фактическим и базовым временем центральной обработки сигнала. Рассчитывается по формуле: ΔE = (t<sub>мышца</sub> − t<sub>корковый</sub>) − t<sub>базовая</sub>.

**M‑ответ** — мышечный ответ при прямой электрической стимуляции нерва. Латентность обычно 5–15 мс в зависимости от длины нерва.

**N20** — корковый соматосенсорный вызванный потенциал для верхних конечностей. Латентность 18–22 мс (с поправкой на рост). Отражает время проведения афферентного сигнала до первичной соматосенсорной коры.

**P40** — корковый соматосенсорный вызванный потенциал для нижних конечностей. Латентность 35–45 мс. Аналогично N20, маркер достижения коры.

**t<sub>мышца</sub>** — латентность M‑ответа (мс).

**t<sub>корковый</sub>** — латентность коркового вызванного потенциала (N20 или P40) в мс.

**t<sub>базовая</sub>** — индивидуальная константа, отражающая нормальное время центральной обработки для данного пациента. Получается при калибровке на здоровом сегменте: t<sub>базовая</sub> = t<sub>мышца</sub> − t<sub>корковый</sub> (в норме).

**CSD (critical slowing down)** — феномен роста автокорреляции и дисперсии временного ряда перед критическим переходом (приступ, ухудшение, потеря сознания). Используется для раннего предупреждения необратимых изменений.

**AR(1)** — автокорреляция первого порядка; один из маркеров CSD. Высокое значение (>0,7) указывает на нарастание инерционности системы.

**Сенсорный ΔE** — модифицированный показатель для пациентов без сознания. Оценивается только t<sub>корковый</sub> (ССВП) без двигательного ответа. Позволяет отслеживать афферентное проведение.

**Латентность** — время от стимула до наступления ответа (в миллисекундах).

**TTL‑триггер** — сигнал синхронизации между стимулятором, ЭЭГ и ЭМГ. Обеспечивает точную временную привязку регистрации к моменту стимуляции.

**ССВП** — соматосенсорные вызванные потенциалы. Метод регистрации электрической активности мозга в ответ на стимуляцию периферического нерва.

**ЭНМГ** — электронейромиография. Метод исследования периферических нервов и мышц.

**ИОМ** — интраоперационный нейромониторинг. Контроль функционального состояния нервных структур во время хирургического вмешательства.
