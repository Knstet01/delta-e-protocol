# Приложение 4. Чек‑лист оборудования

| Компонент | Требование | Статус готовности |
|-----------|------------|-------------------|
| ЭЭГ‑система | 8–16 каналов, ≥ 2000 Гц, синхронизация | |
| Электрический стимулятор | Одиночные импульсы 0,1–0,5 мс, до 50 мА | |
| ЭМГ‑блок | 2–4 канала, накожные/игольчатые электроды | |
| Система синхронизации | TTL‑триггер, задержка < 0,1 мс | |
| ПО для расчёта ΔE | Автоматическое усреднение, AR(1), дисперсия | |
| Термометр кожи | Точность ±0,5 °C | |
| Электроды ЭЭГ | Импеданс < 5 кОм, хлорсеребряные | |
| Электроды ЭМГ | Накожные или игольчатые | |
| Стимулирующие электроды | Поверхностные, 2×2 см, гель | |
